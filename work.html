License Number Reviews

  • Reviews based on the license number plate
  • The user will login via the facebook account (oAuth 2.0).
  • The main dashboard consists of two main tabs: one with the reviews given and the other one with the reviews received for the tracked numbers.
  • Users can write a review based on the license number plate, either negative or positive.
  • The user also has the possibility to track one or more license number plates in order to be informed when those numbers are reviewed.
  • You can search for a license number at any time wanted in order to see what other traffic participans think of a certain driver.
    • Techologies used:
    • Database: MySQL
    • Backend: NodeJS ( Express Framework )
    • Frontend: React JS
Real time Chat Application

  • Real time chat application based on web sockets.
  • Before joining the chat, the user has to register an account.
  • There are two main parts of this application: the Community chat, where all the connected users can talk, and the second part, private messaging between users.
  • Users will be greeted by the system when they are joining and leaving the chat.
  • Typing messages are sent to the chat when a user is typing.
  • Messages are saved in the DB, being loaded when the users re-joins the chat.
  • The messages box is automatically scrolling to bottom when new messages are received.
    • Techologies used:
    • Database: MongoDB
    • Backend: NodeJS ( Express Framework )
    • Frontend: React JS
Car Rental

  • This application offers the possibility to rent a car from a list of cars stored in the database.
  • Car images are stored in database with the help of GridFS ( under base64 format )
  • If the picked date is not available, the user will be notified.
  • In order to make a booking, you have to be logged in.
  • The bookings can be viewed in the ,,Bookings" section when the user is logged in.
  • The user has the possibility to filter the cars by their engine size and narrow the possible rental selection.
  • The login session protection is reached by using JWT ( JSON Web Token )
    • Database: MongoDB
    • Backend: NodeJS ( Express Framework )
    • Frontend: React JS
